# Leaf Disease Classification
A CNN that classifies plant leaf diseases from images, built with PyTorch and trained on the PlantVillage dataset.

## Overview
- **Dataset:** [PlantVillage](https://www.kaggle.com/datasets/abdallahalidev/plantvillage-dataset) (color images)
- **Model:** Custom 3-layer CNN (`LeafDiseaseCNN`)
- **Framework:** PyTorch
- **Environment:** Google Colab (GPU)
